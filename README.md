# Supply Chain Control Tower Dashboard

## 🚀 What is our App?

The **Supply Chain Control Tower Dashboard** is an advanced, enterprise-grade web application designed to provide end-to-end visibility, real-time insights, and predictive modeling for complex supply chain networks. Built for hackathons and production alike, it transforms raw data into actionable intelligence, empowering decision-makers to optimize routes, simulate scenarios, and maximize revenue.

## ⚙️ How does it work?

The platform integrates a robust front-end architecture with real-time backend services and artificial intelligence. 

- **Data Hub:** Connects to a Supabase backend to ingest and manage live supply chain data, from shipment tracking to inventory levels.
- **AI-Driven Intelligence:** Utilizes Google's Gemini AI to analyze data patterns, offering proactive insights and identifying potential bottlenecks before they occur.
- **Interactive Modeling:** Allows users to interact with "Simulation Centers" and "Revenue Models" to test different logistical strategies (e.g., changing shipping routes or evaluating new pricing tiers) and immediately see the financial and operational impact.
- **Dynamic Design System:** Features a highly customizable, "beige tier" enterprise aesthetic. Users can toggle themes, typography, and palettes on the fly via the Settings panel, ensuring a premium user experience tailored to their preferences.

## ✨ Key Features

- **Insights Panel:** AI-powered analytics and alerts for your supply chain operations.
- **Routes Tracking:** Real-time visibility into logistical routes and shipment statuses.
- **Simulations:** Test operational changes and disruptions in a risk-free environment.
- **Snapshots:** Historical data capture for trend analysis and reporting.
- **Revenue Modeling:** Financial projections based on current supply chain performance.
- **Enterprise Aesthetics:** A beautifully crafted, high-contrast UI with editorial typography and dynamic theming.

## 🛠️ Tech Stack

- **Frontend:** React 18, Vite, Tailwind CSS, Lucide React (Icons), Recharts (Data Visualization)
- **Backend & Database:** Supabase
- **Artificial Intelligence:** Google Generative AI (Gemini)
- **Deployment:** Vercel

## 🏃‍♂️ Getting Started

### Prerequisites
Make sure you have Node.js (>= 18.0.0) installed.

### Installation

1. Clone the repository.
2. Install the dependencies:
   ```bash
   npm install
   ```

### Running Locally

Start the Vite development server:
```bash
npm run dev
```
